Projects

Airbnb Market Analysis

The Rent Price Tracker is a data-driven project designed to analyze and visualize rental prices across Vancouver, Burnaby, Surrey, and Richmond. Using web scraping, SQL, and Python (pandas), this tool collects real-time rental data from Rentals.ca, helping users understand market trends and compare prices for different bedroom types.

Café Sales Dashboard

In this project, I analyzed a dataset about sales at a café, focusing on various aspects of the business. The dataset included key information such as product sales, pricing, quantity sold, payment methods (credit or cash), dining preferences (dine-in or takeout), and sale dates. The goal was to clean the data, identify trends, and create insightful visualizations to help the café understand customer preferences and sales patterns.

Secure Transaction Analysis

The Fraud Detection Dashboard is a data-driven web application that analyzes credit card transactions to identify fraudulent activities. Using SQL, Python, and interactive visualizations, the site provides insights into transaction patterns, fraud trends, and customer behavior.
